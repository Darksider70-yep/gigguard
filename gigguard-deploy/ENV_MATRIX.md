## Security Flags
- **FLAG**: `RAZORPAY_KEY_SECRET` found in `backend/src/config.ts`. **ACTION REQUIRED**: Remove from backend.
- **FLAG**: `JWT_SECRET` should not be hardcoded. **ACTION**: Use `generateValue: true` in `render.yaml`.
- **FLAG**: `DATABASE_URL` found in `.env` (local). **ACTION**: Use `fromDatabase` in `render.yaml`.
